üîÆ AI Predictions

Forecasts crowd size for the next 5-10 seconds.

  • CURRENT: Live count right now
  • +5 SECONDS: Predicted count 5 seconds from now
  • +10 SECONDS: Predicted count 10 seconds from now
  • TREND: INCREASING, DECREASING, or STABLE
  • CONFIDENCE: Reliability of prediction (0-100%)

üåä Surge Probability

  • 60-100%: High surge - Take preventive action

Setting Physical Area (Most Important!)

The physical area is the most critical setting. It determines how density is calculated.

Formula:
Density = People Count √∑ Physical Area (m¬≤)
How to Measure:
  1. Identify the area your camera covers
  2. Measure length and width in meters
  3. Multiply: Area = Length √ó Width
  4. Example: 10m √ó 10m = 100 m¬≤

üîß Troubleshooting

Dashboard Offline / Not Loading

Problem: "Cannot connect to localhost:5000"

Solutions:

  1. Check if python app.py is running
  2. Verify no other app uses port 5000
  3. Try: http://127.0.0.1:5000
  4. Check firewall settings
  5. Restart the application

Predictions Show 0

Problem: Predictions not updating

Solutions:

  • Wait 10+ seconds after starting (system needs data)
  • Check browser console (F12) for errors
  • Verify crowd_predictor.py exists
  • Refresh page (Ctrl+F5)
  • Restart application
